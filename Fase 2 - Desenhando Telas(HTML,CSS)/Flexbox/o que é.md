### O que é Flexbox?

- Flexbox, ou Flexible Box Layout, é um modelo de layout do CSS3 que facilita a criação de layouts responsivos e adaptáveis. Ele permite distribuir espaço entre os itens em um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico.

- Com o Flexbox, você pode alinhar e distribuir os itens de forma eficiente, tanto na direção principal (horizontal ou vertical) quanto na direção transversal. Isso torna o Flexbox uma ferramenta poderosa para criar layouts modernos e flexíveis na web.

- O Flexbox é especialmente útil para criar layouts que precisam se ajustar a diferentes tamanhos de tela, como em dispositivos móveis, tablets e desktops. Ele simplifica o processo de alinhamento, espaçamento e ordenação dos elementos dentro de um contêiner.

- Para usar o Flexbox, você define um contêiner flexível aplicando a propriedade `display: flex;` ao elemento pai. A partir daí, você pode usar várias propriedades do Flexbox para controlar o layout dos itens filhos, como `justify-content`, `align-items`, `flex-direction`, entre outras.

- Em resumo, o Flexbox é uma ferramenta essencial para desenvolvedores web que desejam criar layouts responsivos e adaptáveis de forma eficiente e eficaz.
