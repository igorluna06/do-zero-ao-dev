### O que é HTML?

- HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar e estruturar páginas na web. Ele define a estrutura do conteúdo, permitindo que navegadores interpretem e exibam textos, imagens, links, vídeos e outros elementos de forma organizada.

- HTML utiliza "tags" (etiquetas) para marcar diferentes partes do conteúdo. Essas tags são usadas para definir elementos como títulos, parágrafos, listas, links, imagens e muito mais. Cada tag tem uma função específica e pode conter atributos que fornecem informações adicionais sobre o elemento.

- Um documento HTML é composto por uma série de elementos aninhados, começando com a tag `<html>`, seguida pelas seções `<head>` (que contém metadados e links para recursos externos) e `<body>` (que contém o conteúdo visível da página).

- HTML é frequentemente usado em conjunto com outras tecnologias web, como CSS (Cascading Style Sheets) para estilização e JavaScript para interatividade, formando a base para o desenvolvimento de sites e aplicações web modernas.

### É uma linguagem de programação?

- HTML não é uma linguagem de programação; é uma linguagem de marcação. Isso significa que HTML é usado para estruturar e organizar o conteúdo, mas não possui lógica ou funcionalidades de programação, como loops ou condições. Para adicionar interatividade e funcionalidades dinâmicas a uma página web, é necessário usar linguagens de programação como JavaScript.
