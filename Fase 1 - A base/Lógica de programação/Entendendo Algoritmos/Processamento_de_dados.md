### O que é processamento de dados

Entrada -> Processo/Processamento -> Saída

Entrada

- Porta dos dados

Processo/Processamento

- Recebe os dados da entrada
- Verifica o tipo de dado
- Manipula os dados(Calculos, modificar dado, etc..)

Saída

- Gera o resultado das manipulações

### Diferença de dado e informação

Dados

É uma representação de fatos, conceitos ou instruções de maneira formalizada
que devem ser adequados para comunicação, interpretação, ou de transformação
pelo homem ou máquina electrónica

Informação

São organizadas ou dados classificados que tem alguns valores significativos
para o receptor e também são os dados processados no qual as decisões e ações
são baseadas
