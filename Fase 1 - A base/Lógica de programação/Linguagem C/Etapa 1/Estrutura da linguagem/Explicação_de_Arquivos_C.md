### Arquivos C

Quando compilamos um arquivo c, ele retorna 3 tipos de arquivos:

- Arquivo objeto (.o): arquivo binário que contém o código de máquina gerado a partir do código fonte. Ele não é executável por si só, mas pode ser ligado com outros arquivos objeto para criar um executável.

- Arquivo executável: arquivo binário que pode ser executado diretamente pelo sistema operacional. Ele é criado a partir da ligação de um ou mais arquivos objeto.

- Arquivo fonte (.c): arquivo de texto que contém o código escrito em linguagem C. Ele é o ponto de partida para a compilação e deve ser convertido em um arquivo objeto antes de ser ligado para criar um executável.
