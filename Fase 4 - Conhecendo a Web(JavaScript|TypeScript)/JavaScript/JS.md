### JavaScript (JS)

- JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web, tanto no lado do cliente (front-end) quanto no lado do servidor (back-end). Ela é conhecida por sua capacidade de criar interatividade em páginas web, manipular elementos HTML e CSS, e se comunicar com servidores para carregar dados dinamicamente.

- História: JavaScript foi criado por Brendan Eich em 1995 enquanto trabalhava na Netscape Communications Corporation. Originalmente chamado de Mocha, depois renomeado para LiveScript e finalmente para JavaScript, a linguagem rapidamente se tornou uma parte essencial do desenvolvimento web.

- Características Principais:
  - Interpretada: JavaScript é uma linguagem interpretada, o que significa que o código é executado diretamente pelo navegador ou ambiente de execução sem a necessidade de compilação prévia.
  - Dinâmica: JavaScript é uma linguagem dinâmica, permitindo a criação e modificação de objetos e funções em tempo de execução.
  - Orientada a Objetos: JavaScript suporta programação orientada a objetos, permitindo a criação de objetos, herança e encapsulamento.
  - Funcional: JavaScript também suporta programação funcional, permitindo o uso de funções como cidadãos de primeira classe.

- Ecossistema: JavaScript possui um vasto ecossistema de bibliotecas e frameworks, como React, Angular, Vue.js para front-end, e Node.js para back-end, que facilitam o desenvolvimento de aplicações web complexas.

- Uso Comum:
  - Manipulação do DOM: JavaScript é frequentemente usado para manipular o Document Object Model (DOM) de uma página web, permitindo a criação de efeitos dinâmicos, validação de formulários e atualização de conteúdo sem recarregar a página.
  - Comunicação Assíncrona: Com o uso de AJAX (Asynchronous JavaScript and XML) e Fetch API, JavaScript pode se comunicar com servidores para carregar dados em segundo plano, melhorando a experiência do usuário.
  - Desenvolvimento de Aplicações Web: JavaScript é a base para muitas aplicações web modernas, permitindo a criação de interfaces ricas e interativas.

- Exemplos Práticos: Um exemplo clássico de arquitetura cliente-servidor é um site. O navegador (cliente) solicita uma página web do servidor, que processa a solicitação, recupera os dados necessários e envia a página de volta ao navegador para exibição ao usuário.

### Como adicionar JavaScript a uma página Web

1º Forma: Inline

- Inline: Você pode adicionar JavaScript diretamente dentro de uma tag `<script>` no arquivo HTML.

2º Forma: Externo

- Externo: Você pode criar um arquivo JavaScript separado com a extensão `.js` e vinculá-lo ao seu arquivo HTML usando a tag `<script>` com o atributo `src`.
