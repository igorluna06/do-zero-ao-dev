### O que são variaveis

São uma posição na memória da máquina(RAM) que identificamos por um nome (identificador).
Na linguagem C, todas as variáveis precisam ser declaradas no início do programa, 
antes de serem usadas

### Tipo de dados

Palavra Chave | Formato para leitura | Tipo

int             %d                     Inteiro
float           %f                     Real com precisão simples
double          %ld                    Real com precisão dupla
char            %c                     Caractere

### Regras de declaração

- Case sensitive(sensivel em relação a minusculas e maiusculas)
- Nunca começar uma variável com números ou acentos
- Usar underline(_) para variável composta
- Não pode ser declarada
- Nada de usar palavras reservadas
