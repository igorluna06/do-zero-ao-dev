### O que é CSS?

- CSS significa "Cascading Style Sheets" (Folhas de Estilo em Cascata).
- É uma linguagem usada para descrever a apresentação de um documento escrito em HTML ou XML.
- CSS define como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias.

### Por que usar CSS?

- Permite separar o conteúdo (HTML) da apresentação (CSS), facilitando a manutenção e o design.
- Proporciona controle sobre o layout, cores, fontes e outros aspectos visuais de uma página web.
- Facilita a criação de designs responsivos que se adaptam a diferentes tamanhos de tela e dispositivos.

### Como funciona o CSS?

- O CSS é composto por regras que consistem em seletores e declarações.
- Um seletor aponta para o elemento HTML que você deseja estilizar.
- As declarações dentro de chaves `{}` definem as propriedades e valores que serão aplicados ao elemento selecionado.

### O que são seletores?

- Seletores são padrões usados para selecionar os elementos HTML que você deseja estilizar.
- Exemplos de seletores incluem:
    - Seletor de elemento: `p` (seleciona todos os parágrafos)
    - Seletor de classe: `.minha-classe` (seleciona todos os elementos com a classe "minha-classe")
    - Seletor de ID: `#meu-id` (seleciona o elemento com o ID "meu-id")
